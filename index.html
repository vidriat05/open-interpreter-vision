The Open Interpreter Project will build a standard interface between humans, language models, and computers.


I

The Language Model / Computer Interface

We have developed a real-time code execution environment that language models can operate.

Since then, we've started building a high-level computer API that exposes the foundational capabilities of computers to language models.


II

The Human / LMC Interface

Language Model Computers offer fundamentally new modes of human-computer interaction, which we intend to explore.
