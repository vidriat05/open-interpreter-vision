Open Interpreter is an open-source initiative to build a standard interface between humans, language models, and computers.


I

The Language Model / Computer Interface (2024)

We have developed a real-time code execution environment that language models can operate.

Since then, we've started building a high-level computer API for LLMs that exposes the foundational capabilities of computers.


II

The Human / LMC Interface (2025)

Language Model Computers offer fundamentally new modes of human-computer interaction, which we intend to explore.
